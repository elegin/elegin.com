    Lanyard (Base 3)

      1. Cipher

        1. 3 unique lanyards all made up of only 3 unique symbols

        2. Each lanyard is made up of the same number of symbols

        3. The order of symbols differs for each color of lanyard.

        4. By lining the lanyards next to each other in the correct order, and viewing the three symbols as base 3 numbers, we can obtain numbers in the range [0,26]

        5. Lanyard ordering could be determined by Asian symbols at the top of each lanyard (to be ordered as “ten”, ”five”, “seven”) (We used brute force here)

        6. Value to symbol assignments found through brute force.

      2. Plaintext / Riddle

        1. the ox door closer chases the green witch five min behind her he used to be mary

      3. Solution

        1. In oxford, “Great Tom”, which used to be called Mary, rings 5 minutes behind Greenwitch time, keeping with its historic position as a closer bells.
